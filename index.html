Projects

Habitizer

This was a class project where the goal was to practice proper software development strategies used in the workplace. With a team of 6, we created a routine tracking app that allows users to time how long they spend on a task while maintaining great project planning, coordinated sprints, and effective communication.

Pace Partner

I used this project to experiment with learning how to develop a Full-Stack app. I used Node.js and Express.js for the backend which introduced me to proper schema design, database connections, and API calls. The goal of this project was to give users a platform to learn proper paces for running and to plan their workouts throughout the week.

Minesweeper

Minesweeper is a game that I am very fond of so I used this as an opportunity to learn React.js while developing a game that I am very passionate about. I collaborated with a friend to create a fully functional Minesweeper game that simple recreates the original game.
